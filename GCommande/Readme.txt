C'est une plateforme réaliser uniquement en J2EE
+ J'ai utilisé le serveur Tomcat 9, en ajoutant les deux jars dans son rep.
+J'ai utilisé mamp (username=root,password=root) et phpmyadmin 
+ On utilise cet URL "http://localhost:8080/GCommande/admin" pour accéder à la session admin username=admin,password="1234";
dans la session admin l'administrateur peut ajouter ,modifier et supprimer les produits plus l'affichage de la liste des produits.
+ URL qui se lance automatiquement est " http://localhost:8080/GCommande/ " ici on trouve les categories et les produits enregistrés dans la base de donnée on peut les filtrer par prix, ainsi que l'inscription, la connexion et cart qui contient la liste des produits ajoutés au panier avec le prix total des produits selectionné
